# Descrição

Ignite Feed | Rede social.

## Tecnologias

-   React.js (vite)
-   TypeScript
-   Phosphor React (icons)
-   Date fns

## Como utilizar?

**Clone o repositório**

```bash
$ git clone https://github.com/igorct1/dogs-ts.git](https://github.com/igorct1/01-fundamentos-reactjs.git
```

**Instale as dependências**

```bash
$ npm install
```

**Rode o projeto**

```bash
$ npm start
```
